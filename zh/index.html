🐾Petfolio 能做什么

🐾

每一只宠物,都有属于自己的档案

  • 支持 8 个物种:狗、猫、兔、鸟、小型哺乳动物、鱼、爬行动物。鱼、爬行动物和"其他"使用自由输入品种。
  • 5 个物种下 477 个品种,带有 AKC / TICA / ARBA 分组及按物种划分的体型分级。通用「混血 / 家养」条目兜住没有血统的宠物。
  • 档案中枢:主照片、年龄、体重,以及 3×3 快捷方格,直达健康、体重、兽医、照片、食物、生命里程碑、支出、保险与紧急联系人。
  • 每只宠物无限照片,保留原始 EXIF 信息。
💚

健康记录,细致而温和

  • 体重 — 折线图带有上升 / 下降 / 稳定趋势,支持多宠物对比。
  • 疫苗与驱虫(内驱 / 外驱 / 综合)— 附带逾期、今日、即将到期的状态徽章。
  • 用药 — 计划 / 记录分离,处方变更后记录依然保留;支持按间隔或按需服用,列表一键记录。
  • 症状与洗澡 — 自由文字记录,自动汇入时间线。
  • 护理日志 — 倒序时间线,支持类型筛选;可撤销的左滑删除。
📅

日历与提醒

  • 列表视图显示逾期、今日、未来事项;月视图一览全貌。
  • 通知最多可提前 7 天,角标显示逾期与今日数量。
  • 聚合疫苗、驱虫、用药、兽医复诊、重要日子、保险续费等事件。
🥣

资源标签页 — 把家中所有目录集中起来

  • 跨宠物聚合支出、食物、营养(保健品)、活跃用药、猫砂、垫材与兽医联系人。
  • 目录条目可绑定到单只宠物,也可作为家庭级条目 — 家庭级条目随家庭共享一同同步。
  • 食物、营养、猫砂均带产品照片、评分、价格与可点开的详情页。
  • 保险费按账期分摊、跨年时按天数划分,自动汇入支出。
🏥

兽医就诊与保险

  • 就诊记录可附加 PDF 和图片 — QuickLook 预览,从「文件」或「照片」导入。
  • 应用内共享的兽医联系人目录,详情可复制并一键拨号、邮件、地图。
  • 按宠物管理保险单,续费日期会出现在日历与提醒中。
📷

照片与 Live Photo

  • 每只宠物可保存多张静态照片与实况照片(Live Photo) — 成对的 JPG + MOV 存储,应用内播放,导入导出全程保留。
  • 主体剪影头像 — 宠物轮廓叠加在圆形头像上,长按时随 Live Photo 一起动起来。
✨

智能小帮手

  • Gemini AI 文本提取 — 拍摄兽医记录、疫苗标签、药瓶或驱虫产品,在保存前确认提取内容。覆盖全部 7 类事件。可选功能,需自备免费 Gemini API 密钥,安全存放在 iOS 钥匙串中。
  • 主屏小组件 — 「即将到来」与轮播的「宠物照片」。
  • Spotlight 搜索 — 从主屏即时按名字或品种找到任意宠物。
💾

你的数据,归你所有

  • 所有数据只留在你的设备或 iCloud 中 — 我们不运行任何服务器,也从不查看你的记录。
  • 完整 ZIP 备份(含主题头像与各条目照片),恢复时可合并或替换;含 Zip Slip 保护与删除前验证。
👨‍👩‍👧‍👦

与家人共享这个小家

家庭共享让所有照顾宠物的人都能看到相同的记录、提醒和照片。一次轻点,即可邀请伴侣、室友或寄养人,通过 iCloud 让全家同步无缝。
